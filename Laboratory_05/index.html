HTTP

  1. The hypertext transfer protocol is a high level protocol for sharing hypertext on the web. It was design to be part of the Internet Protocol and depends on a reliable transport layer protocol(TCP,UDP,SSDP,UDP). It is based on the Representational state transfer (REST) architecture.

  2. It has two mayor versions 1.0 and 1.1.

  3. The way information passed to the server is with the utilization of "methods" each having a unique purpose for the server.

    • HTTP 1.0 has:
      • GET
      • POST
      • HEAD
    • HTTP 1.1 added:
      • PUT
      • UPGRADE,
      • ...

GET Method

This method is intended for information fetching and it passes parameter to the server via the URL.


Example Request


Request Line -> GET / HTTP/1.0
                Host: google.com

Example Response


					HTTP/1.0 200 OK
					...

POST Method

This method is intended for the passing of information to the server that it will permanently keep and stored.


Example Request


Request Line -> POST / HTTP/1.0
                Host: google.com
                Content-Type: application/x-www-form-urlencoded
                Content-Length: length

                param1=value1&;param2=value2

Example Response


					HTTP/1.0 200 OK
					...
